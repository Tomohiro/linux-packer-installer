Linux Packer Installer
================================================================================

What is this
--------------------------------------------------------------------------------

Download [Packer](http://packer.io) and install tool for Linux.

I've tested on Ubuntu 12.04 LTS.


Commands
--------------------------------------------------------------------------------

Install Packer:

```sh
$ ./linux-packer-installer install
```

Uninstall Packer:

```sh
$ ./linux-packer-installer uninstall
```

Show usage:

```sh
$ ./linux-packer-installer --help
```
